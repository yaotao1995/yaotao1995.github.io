总结下这段时间干了些什么

  • 主要包括自己写的项目、学习等

1、复习 - OC和C实现基础算法

复习基础算法，使用OC 和 C 写了基础算法的实现，简单的分析了优劣
包括：插入、冒泡、鸡尾酒排序、二分查找、快速排序、希尔排序
项目上传GitHub了

2、关于数学课本项目的上架失败问题

最近为了复习数学知识，写了一款可以在线看书的APP，主要功能：

  • 1.用户可以在线打开pdf
  • 2.用户可以下载pdf 方便无网络浏览
  • 3.可以保存阅读进度，重新打开跳转到上一次看的位置

但是在用自己个人开发者账号上架时候碰到的问题大概如下：

Guideline 4.2 - Design - Minimum Functionality

  • Books should be submitted to the Apple Book Store. To work with Apple on Book Store distribution, you should first verify that your content meets the following requirements:
  • ISBNs are required for all paid titles you intend to distribute
  • Is in either:
  • EPUB format, passing EpubCheck 1.0.5
  • Multi-Touch format, created by iBooks Author
  • 大体意思就是这个APP并没用合理的用户交互功能（只是拿来看的），书籍应该提交到苹果书店
  • 所以就GG了

3、逆向优酷去掉广告的问题

  • 近几天，本着友好学的目的，尝试去掉优酷的广告，广告是去掉了，但是在打开试看视频的时候，发现广告去掉了，但是试看的6分钟限制还在

分析过优酷的代码发现个事情，优酷有 YTCoreSoftwarePlayerView这么个核心播放类 和 XAdEnginePreAdModule 广告的类

  • 普通视频在播放的时候直接将广告类置空就可以看了
  • 问题在于，试看视频，应该是维护了6分钟的倒计时，时间一到直接暂停核心播放类，所以去掉了广告层是没用的
  • 于是乎我想尝试打印这个类的所有方法调用，想知道6分钟限制到底是在哪里
  • 但是在make install 之后，重新打开app就奔溃了，还没来及看奔溃日志，就忙项目去了，等有时间再继续搞吧
